- 应用名称从"儿童识字乐园"改为"学习乐园"

  - 📚 儿童识字模块（学习看板、汉字库、开始学习、周统计、我的奖励）

## 升级指南

```bash
docker compose up -d --build
```

### v1.0 → v2.1
参考 `快速升级指南.md` 或运行：
